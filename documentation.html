General

Prefix

You can change prefix of all class added on elements of Trumbowyg using this option : 


$('textarea').trumbowyg({
    prefix: 'custom-prefix'
});
For example, the bold button class is now custom-prefix-bold-button

Default value is "trumbowyg"

Localization

You don't speak english? No problem, Trumbowyg has a language parameter. It's necessary to load your lang file. Search in /trumbowyg/langs folder if a language file already exist, else create and share it :). Go to Add a localization part for more details on language file.

Don't forget include the lang file in your pages : 


<script type="text/javascript" src="js/trumbowyg/langs/fr.js"></script>

Warning : Include lang file after Trumbowyg and before instantiate the editor !

Usage of language parameter : 


$('textarea').trumbowyg({
    lang: 'fr'
});

Note : If the lang was not found, english values are used by default.

Direction

If your language needs to be read from left to right, do this with the dir option : 


$('textarea').trumbowyg({
    dir: 'rtl'
});

Animation duration

Lot of elements have animations, and duration of animations is configurable via the duration options : 


$('textarea').trumbowyg({
    duration: 1000
});

Note : duration is defined in milliseconds, because this value is directly used in .animate() jQuery method.

Add a localization

The structure of a language file is simple, it is a JavaScript Object which associate a translation to any key. For example : 


$.trumbowyg.lang.fr = {
    bold: "Gras",
    close: "Fermer"
};

API

Modal box

When you want create your custom extension for Trumbowyg, you can open and close a modal box with custom inner HTML code. For that use the .trumbowyg() method and give parameters like that : 


// Open a modal box
$('textarea:nth-child(2)').trumbowyg('openModal', {
    title: "A title for modal box",
    fields: {
        nbRows: {
            title: "Number of rows"
            // or $('textarea:nth-child(2)').trumbowyg('opt').lang.nbRows;
        },
        nbCols: {
            title: "Number of cols",
            value: 5
        }
    }
});

// Close current modal box
$('.close-modal').click(function(e){
    $('.trumbowyg').trumbowyg('closeModal');
    e.preventDefault();
});
