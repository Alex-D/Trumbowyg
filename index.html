Introducing Trumbowyg

Why you should love it!

HTML5 ready

Editor and generated code are optimized for HTML5 support. Is compatible with all recents browsers like IE8+, Chrome, Opera and Firefox.

Fast & lightweight

All existing WYSIWYG editors are larger than 45kB. Trumbowyg is only 15kB for faster page loading. No useless features, just the necessary ones to generate a clean and semantic code.

Customizable

Options and design are fully editable and configurable, but the default design is compatible with Retina display and optimized for a great and simple user usage.

Getting Started

It's simple but still I will explains

This the basic code for transform a simple div to an amazing WYSIWYG editor who's Trumbowyg. 


$('#trumbowyg-demo').trumbowyg();

If you want set options to Trumbowyg, add an object who contains your options as parameter. 


$('#trumbowyg-demo').trumbowyg({
    fullscreenable: false,
    closable: true,
    btns: ['bold', 'italic', '|', 'insertImage']
});

View all options in documentation page.

Customization

CSS is cool, make your own Trumbowyg skin!

Trumbowyg is souple and simple. You want to set another skin ?

Replace existing CSS file

Copy /trumbowyg/design folder everywhere and customize style. Finaly, link your new CSS file and remove link to Trumbony default style.

Note : It is not recommended to edit this files in the trumbowyg folder, because when you update plugin, new files errase your modifications.


Multiple skins for multiple Trumbowyg

It is possible to prefix all class used by the editor. Indeed, Trumbowyg integrates an option named prefix. That is usefull when you do not want same design for all instances of Trumbowyg.

Usage of the prefix option : 


$('.editor-modern-ui').trumbowyg({
    prefix: 'modern-ui'
});
In your CSS, you can now stylize .modern-ui-link-button for example, and sure do not have conflicts with default skin. It's necesary to redefined style for all elements. If you want to start from the default skin, copy /trumbowyg/design folder, replace all trumbowyg- by your prefix and work on it.

Note : Default value of prefix is "trumbowyg".
