Introducing Trumbowyg

Why you should love it?

HTML5 ready

Editor and generated code are optimized for HTML5 support. Compatible with all recents browsers like IE8+, Chrome, Opera and Firefox.

Fast & lightweight

All existing WYSIWYG editors are larger than 45kB. Trumbowyg is only 15kB which means faster page loading. No useless features, just the necessary ones to generate a clean and semantic code.

Customizable

Options and design are entirely configurable ot suit your needs. However, the default design is compatible with Retina display and optimized for a great and simple user experience.

Getting Started

As straightforward as it gets

Installation

First, if you don't already do it, load jQuery at bottom of <body> like so: 


<script src="//ajax.googleapis.com/ajax/libs/jquery/1.10.2/jquery.min.js"></script>
<script>window.jQuery || document.write('<script src="js/vendor/jquery-1.10.2.min.js"><\/script>')</script>

Trumbowyg needs jQuery >= 1.7, because it uses .on() a new function appeared in version 1.7.

After this lines, you have to load Trumbowyg. 


<script src="js/vendor/trumbowyg.min.js"></script>

Don't forget to load Trumbowyg CSS in the <head>, or load your own style for the editor. 


<link rel="stylesheet" href="css/vendor/trumbowyg.css">

Basics

This the minimal code to transform a simple div into the amazing WYSIWYG editor which is Trumbowyg. 


$('#trumbowyg-demo').trumbowyg();

If you want to set options to Trumbowyg, add an object which contains your options as parameters. 


$('#trumbowyg-demo').trumbowyg({
    fullscreenable: false,
    closable: true,
    btns: ['bold', 'italic', '|', 'insertImage']
});

View all options in the documentation page.
