Speech recognition plugin

Basic usage

This plugin allow you to enter text via speech recognition in Trumbowyg.

Read speech recognition plugin documentation

The code


$('#editor')
.trumbowyg({
    btns: [
        ['speechrecognition']
    ]
});

Language setting

The code


$('#editor-settings-lang')
.trumbowyg({
    btns: [
        ['speechrecognition']
    ],
    plugins: {
        speechrecognition: {
            lang: 'de-DE'
        }
    }
});

In head tag

At the end of body
