Preformatted plugin

Basic usage

Wraps/unwraps your code with <pre><code> tags.

Read preformatted plugin documentation

Some code to wrap

$('#editor') .trumbowyg({ btns: [ 'viewHTML', 'preformatted' ] });

Then check the HTML, code is now wrapped into pre + code tags!

The code


$('#editor')
.trumbowyg({
    btns: [
        'viewHTML',
        'preformatted'
    ]
});
